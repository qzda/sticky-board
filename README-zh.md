# [Sticky Board](https://sticky-web.netlify.app/)

一个简洁优雅的网页便签板应用，支持拖拽创建、移动、调整大小和删除便签。

## ✨ 特性

- 🎨 **拖拽创建** - 在空白处按住鼠标拖拽即可创建便签，实时预览阴影
- 📝 **自由编辑** - 每个便签都包含可编辑的文本区域
- 🔄 **拖拽移动** - 点击便签顶部拖动到任意位置
- 📏 **调整大小** - 拖拽右边缘和底部边缘调整便签尺寸
- 🗑️ **删除功能** - 点击便签右上角的删除按钮，二次确认后删除
- 💾 **自动保存** - 所有操作自动保存
- 🌓 **深色模式** - 自动适配系统主题
- 📐 **网格对齐** - 所有操作自动对齐到网格
- 🔒 **数据隐私** - 所有数据仅存储在浏览器本地

## 📖 使用说明

### 创建便签

1. 在空白区域按住鼠标左键
2. 拖拽到合适的大小（会显示虚线预览框）
3. 松开鼠标创建便签
   - 注意：宽度和高度都需要至少 10rem (160px)
   - 如果拖拽范围太小，松开后会自动取消

### 编辑便签

- 点击便签内的文本区域即可编辑内容
- 内容会自动保存

### 移动便签

- 点击便签顶部的灰色区域（非文本区域）
- 拖拽到目标位置
- 会自动对齐到网格

### 调整大小

- 拖拽便签的右边缘或底部边缘
- 可以同时拖拽右下角调整宽度和高度

### 删除便签

- 点击便签右上角的 "×" 按钮
- 在确认对话框中点击"确定"
