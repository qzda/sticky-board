# [Sticky Board](https://sticky-web.netlify.app/)

[English](./README.md) | [中文](./README-zh.md)

一个简洁优雅的网页便签板，用于可视化思考，支持拖拽操作和 Markdown 语法。

所有数据仅保存在本地，无需账号，无需同步，打开即用。

## ✨ 特性

- 🔒 **本地存储，保护隐私** – 所有数据仅存储在浏览器中
- 📄 **Markdown 支持** – 使用简单的 Markdown 语法编写内容
- 🎨 **拖拽创建** – 在空白区域拖拽即可创建便签
- 🔄 **拖拽移动** – 拖动便签顶部即可自由移动
- 📏 **调整大小** – 拖拽边缘或角落调整便签尺寸
- 📐 **网格对齐** – 所有操作自动吸附到网格
- 🗑️ **删除便签** – 删除时提供确认，避免误操作
- 💾 **自动保存** – 所有修改都会自动保存
- 📤 **导入 / 导出** – 便签可导入或导出为 JSON 文件
- 🌓 **深色模式** – 自动跟随系统主题
- 🌍 **多语言** – 根据浏览器语言自动切换中英文

## 📖 使用说明

### 创建便签

1. 在空白区域按住鼠标
2. 拖拽确定大小
3. 松开鼠标即可创建
   - 最小尺寸：10rem × 10rem（160px）

### 编辑便签

- 点击右上角 “+” 进入编辑模式
- 使用 Markdown 编写内容
- 点击 “✓” 完成编辑

### 移动与调整大小

- 拖拽便签顶部即可移动
- 拖拽边缘或右下角调整大小
- 所有操作都会自动对齐到网格

### 删除便签

- 点击右上角 “×”
- 确认后删除

### 导入 / 导出

- 点击右下角 **设置** 打开选项
- **导出**：保存为 `stickys-[时间戳].json`
- **导入**：从 JSON 文件导入
  - 相同 ID：更新
  - 新 ID：新增
  - 缺失 ID：保留现有便签
